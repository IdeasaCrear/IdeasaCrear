EJEMPLOS DIGITALES

PAGINAS WEB

Las páginas web pueden contener información o interaccion dirijida a los usuarios, como un formulario, cambios en su perfil, compra de productos, recibir pagos rapidos y seguros, etc. Con las páginas web puedes mostrar información de tu proyecto, localizacion de negocio, mostrar tus productos, curriculum, página de aterrizaje y que representen la imagen de tú proyecto, desde el color de fondo hasta lo que pueden hacer los usuarios en tú página web.

SITIOS WEB

Los sitios web son creados con dos o más páginas web, al separarlo por secciones en el menu de navegacíon, cada página sera una seccion del sitio como por ejemplo pagos, formulario para registrarse, información, archivos a descargar, seccion de preguntas frecuentes, etc. Esto hace más facil para sus usuarios el navegar y saber como esta organizado el sitio de forma muy sencilla y rapída.

APP MOVIL

Una app movil puede ser muy simple o compleja por la cantidad de contenido, interaccion del usuario o diseño de la app, las siguientes son 3 formas posibles de como puede funcionar su app movil:

  1. Todo lo que necesita para funcionar desde la primera instalación y solo requiere internet para actualizarlo, necesita más almacenamiento en el telefono.
  2. Recibe gran parte de la informacion o requiere totalmente de internet para funcionar, la aplicacion requiere menos almacenamiento y debes estar conectado todo el tiempo a la red.
  3. Tambien es posible combinar las opciones anteriormente mencionadas, dejando secciones como registrarse, iniciar sesion o enviar un comentario como opcionales y que solo esas secciones requieran internet para funcionar.
